# mposter

A simple poster app built with Node.js, Express, and MongoDB.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Invia-shubham/mPosterApi.git
   ```

2. Navigate to the project folder:

   ```bash
   cd mposter
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the server:

   ```bash
   npm start
   ```

2. Visit `http://localhost:3000` in your browser.

3. Visit `http://localhost:3000/api-docs` in your browser for api docs.

## Dependencies

- **bcryptjs**: Password hashing.
- **express**: Web framework for Node.js.
- **jsonwebtoken**: For secure authentication.
- **mongoose**: MongoDB object modeling.
- **multer**: Middleware for file uploads.
- **swagger-jsdoc**: Swagger API documentation generation.
- **swagger-ui-express**: Swagger UI integration for Express.
